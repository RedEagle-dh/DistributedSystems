## Compiling

Es sind keine weiteren Schritte zum kompilieren nötig. 

Zum kompilieren von z.B. `ProcessFile.c` folgendes ausführen:

* `gcc ProcessFile.c -o ProcessFile`

Zum ausführen:

* `./ProcessFile`
